# jumentosec.lol

This is the source code for  JumentoSEC website, written in Next.JS: [jumentosec.lol](https://jumentosec.lol)

## Commands

Develop:
```
npm run dev
```

Build production:
```
npm run build
```

Serve production:
```
npm run start
```
